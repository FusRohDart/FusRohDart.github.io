
--------------------------------------------------------------------------
Instructions
--------------------------------------------------------------------------

> Extract contents into a folder.
> Open the folder using Visual Studio Code, or another code editor.
> I recommend using git bash as the terminal to install Node.js and 
all the npm packages. However, anything should be fine. 
> Install Node.js to install packages via npm. 
> Using the terminal in your code editor, generate a package.json file 
using the following commmand; npm init or npm init -y 
> Install the following packages via npm: 
  - bcrypt
  - body-parser
  - cors
  - express
  - express-jwt
  - joi
  - jsonwebtoken
  - mysql2
  - nodemon
  - rootpath
  - sequelize 
> Make sure you have a MySQL Server instance running. Open the Services 
Windows app to make sure the MySQL service is running. 
> Update your MySQL login details in dbConfig.JSON. 
> To run the project, type in the following command in the terminal; 
nodemon restServer.js 
> Use postman to send requests to the server. 
> Once you register and login, use the token generated to access other 
functions. Copy/paste the token value to the token bearer field in the 
application section of the request.
